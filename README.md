# Balance-H-Global
Herramienta de Hydro-Intelligence para balances hídricos globales. Usa Google Earth Engine (GEE) y Python para procesar series CHIRPS (0.05°) y ERA5-Land. Implementa un buffer de 5.5 km de diámetro para coincidir con la resolución nativa del píxel (Escala 1:1), garantizando precisión en el cálculo de P, ETP y Huella Hídrica.
