# Global Hydro-Intelligence Tool
Geospatial Engine for Water Balance & Sustainability Metrics
Esta herramienta de código abierto permite el análisis hidro-climático de alta precisión en cualquier punto del planeta. Diseñada para profesionales en ingeniería ambiental y sostenibilidad, automatiza la obtención de indicadores clave como la Huella Hídrica Verde y balances hídricos multitemporales.

  
*Precipitación: Fuente CHIRPS V2.0 - Descripción Técnica Resolución 0.05° (~5.5 km)*

*Temperatura/ETP - Fuente ERA5-Land - Descripción Técnica Resolución 0.1° (~11 km)*

*Buffer Espacial - Fuente Geometría Circular - Descripción Técnica Diámetro de 5.5 km para escala 1:1 con el píxel*
